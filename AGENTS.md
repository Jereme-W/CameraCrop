## Nuke script workflow

- This repository is being used to iterate on a Nuke script that will often be copy-pasted directly into Nuke for testing before any PR is created.
- After modifying the main `.py` script, always include the COMPLETE updated contents of that file in a single fenced code block.
- Put the full updated script first, then any explanation after it.
- This script is being written specifically for Nuke 14.1v5, but scripts should be able to work in versions 15 & 16 as well. It is meant to use the classic 3D system in Nuke.
- FBX files are being generated in the ASCII FBX 2020 standard.
